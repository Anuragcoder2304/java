# Java-Basics

Here we do basics of java for concept clear.

## Requirements

- **Java JDK**: Version 11 or higher
- **IDE**: IntelliJ IDEA, Eclipse, or Visual Studio Code

## How to Run

1. Open your terminal or command prompt.
2. Compile the Java file:
   ```bash
   javac [FileName].java
   ```
3. Run the program:
   ```bash
   java [FileName]
   ```
